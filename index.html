üìñ Usage Instructions

For Model Access:

  • All assets are organized in subfolders: characters/, objects/, and sensory/
  • File names follow a clear naming convention for easy programmatic access
  • Characters: smiling-boy.svg, excited-boy.svg, walking-boy.svg, laughing-boy.svg, pointing-boy.svg, standing-boy.svg
  • Objects: ball.svg, car.svg, book.svg, robot.svg, teddy-bear.svg, apple.svg
  • Sensory: board.svg, window.svg, door.svg, desk.svg, chair.svg, sun.svg, cloud.svg, tree.svg, slide.svg, bench.svg, path.svg, house.svg, roof.svg, chimney.svg

Color Customization:

  • All SVGs use currentColor and opacity for easy color customization
  • You can change colors by setting the color CSS property on the SVG element
  • Example: <img src="assets/characters/smiling-boy.svg" style="color: #FF5733;">
  • Or embed directly: <svg style="color: #FF5733;">...</svg>
